# Code of Conduct

This project follows a simple contributor covenant:

- Be respectful and constructive.
- Assume good intent, but keep feedback concrete.
- Do not post secrets, private logs, tokens, cookies, or personal data.
- Keep issues focused on reproducible behavior, documentation gaps, or small improvements.
- Maintainers may close or edit comments that are abusive, spammy, or unsafe.

For security-sensitive reports, use the process in [SECURITY.md](SECURITY.md) when that file exists.
